# Ray Cyber Resume Site

这是一个纯静态赛博朋克风个人简历网站。

## 本地预览

直接双击 `index.html` 即可打开。

也可以用 VS Code 的 Live Server 插件预览。

## 修改内容

主要改 `index.html` 里的文字即可。

## 替换 PDF 简历

把你的 PDF 简历放到同目录，命名为：

```txt
resume.pdf
```

首页“下载PDF简历”按钮就能正常打开。

## 部署到 GitHub Pages

1. 把这些文件复制到你的 GitHub 仓库根目录。
2. 提交并 push。
3. GitHub 仓库进入 Settings → Pages。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待生成网址。
